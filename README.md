# promptedit-home

Homepage for [promptedit.com](https://promptedit.com)
## Stack

| | |
|---|---|
| Framework | Vue 3 + TypeScript |
| Build tool | Vite 7 |
| Styling | Tailwind CSS v4 |
| Linting | ESLint 10 (flat config) |
| Formatting | Prettier + prettier-plugin-tailwindcss |

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lint

```bash
npm run lint
```
